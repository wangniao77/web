高校发展治理平台目标态产品架构

最终产品完整架构：学生数据为现有基础，外部系统逐步接入；智能助手内嵌 Student 学生成长画像

现有可直接使用的数据

  • • 22-25级学籍成绩 Excel：学号、姓名、年级、专业、班级、GPA、课程均分、学分、缺考补考重修、四六级、竞赛、班主任、辅导员
  • • 可直接支撑学生成长画像、学院学生画像、学业预警、高潜识别、部分教学质量与竞赛成果聚合

目标态需外部接入的数据

  • • 教务、学工心理、就业升学、科研竞赛证书、治理任务、预算财务、排名对标、视频物联等系统
  • • 外部数据补齐校级竞争力、双一流建设、就业去向、心理风险、科研经费、社会服务和视频分析

Student 内嵌智能助手

  • • 财宝助手读取学生面板数据，构建结构化上下文后调用 LLM
  • • LLM 输出亮点分析、需关注事项、匹配依据、短期规划、长期规划和就业建议，并沉淀到报告中心

# 数据导出说明

## 前端（任一页面）

| 入口 | 位置 |
|------|------|
| 学生总览 | 顶栏「导出Excel」 |
| 学生二级页 | `StudentDetailLayout` 标题栏右侧 |
| 学院总览 | 顶栏「导出Excel」 |
| 学院明细页 | `CollegeDetailLayout` 标题栏右侧 |

### 导出优先级

1. 页面通过 `usePageExport` / `useStudentDashboardExport` 注册的专用数据
2. 否则：页面内 HTML `table`（`collectDomTables`）
3. 学生页再兜底：拉取 dashboard 快照（基本信息 / 学业 / 奖学金 / 预警 / 竞赛）

实现：`src/utils/exportExcel.ts`（SpreadsheetML `.xls`，Excel 可直接打开，无 xlsx 依赖）

## 后端接口

- `GET /student/{student_id}/export.xlsx`
- 实现：`Utils/Excel/export_workbook.py`（openpyxl）

## 小表设计

见 `docs/数据库小表设计.md` 与 `docs/sql/small_tables_dict_scholarship.sql`。
库未建表时需先执行该 SQL（或 Aerich 迁移），奖学金字典与事实表才会有数据。
